# Employee Offboarding

## Notification and Planning
- [ ] Receive and document resignation or termination
- [ ] Determine last working day
- [ ] Notify HR and begin offboarding process
- [ ] Notify IT to schedule account deactivation
- [ ] Plan knowledge transfer timeline
- [ ] Schedule exit interview

## Knowledge Transfer
- [ ] Document ongoing projects and status
- [ ] Identify replacement or interim owner for tasks
- [ ] Transfer ownership of shared documents and files
- [ ] Record key contacts and vendor relationships
- [ ] Update team documentation and runbooks
- [ ] Hand over passwords and shared credentials to manager

## Access and Accounts
- [ ] Revoke building and badge access
- [ ] Disable email account
- [ ] Remove from Slack, Teams, and communication tools
- [ ] Revoke VPN and remote access
- [ ] Remove from software licenses and SaaS tools
- [ ] Transfer ownership of shared drives and repositories
- [ ] Remove from distribution lists and groups

## Equipment
- [ ] Collect laptop and charger
- [ ] Collect mobile phone if company-issued
- [ ] Collect security badge and keys
- [ ] Collect parking pass
- [ ] Collect any other company property
- [ ] Wipe company data from personal devices if BYOD

## Financial and Benefits
- [ ] Process final paycheck including unused PTO
- [ ] Provide COBRA or benefits continuation information
- [ ] Address stock options or equity vesting
- [ ] Process expense report for outstanding reimbursements
- [ ] Update payroll system

## Wrap-Up
- [ ] Conduct exit interview
- [ ] Update org chart and team directory
- [ ] Communicate departure to team and stakeholders
- [ ] Archive employee records
- [ ] Send farewell message or announcement
- [ ] Provide reference letter if appropriate
