# Monthly Personal Finance Review

## Income
- [ ] Verify all paychecks deposited correctly
- [ ] Record any freelance or side income
- [ ] Note any reimbursements received
- [ ] Update income tracking spreadsheet

## Bills and Expenses
- [ ] Review all credit card statements
- [ ] Check for unauthorized charges
- [ ] Verify recurring subscriptions are still needed
- [ ] Pay all bills before due dates
- [ ] Review utility usage for unusual spikes
- [ ] Categorize all expenses

## Budget Check
- [ ] Compare actual spending to budget by category
- [ ] Identify areas of overspending
- [ ] Adjust next month's budget if needed
- [ ] Review discretionary spending
- [ ] Calculate savings rate for the month

## Savings and Investments
- [ ] Confirm automatic savings transfers occurred
- [ ] Review emergency fund balance
- [ ] Check retirement account contributions
- [ ] Review investment portfolio performance
- [ ] Rebalance portfolio if needed
- [ ] Contribute to any sinking funds (vacation, car, etc.)

## Debt
- [ ] Review all outstanding balances
- [ ] Confirm minimum payments were made
- [ ] Make extra payments if possible
- [ ] Track progress toward payoff goals
- [ ] Check interest rates for refinancing opportunities

## Credit
- [ ] Check credit score
- [ ] Review credit report for errors
- [ ] Dispute any inaccuracies
- [ ] Monitor for signs of identity theft

## Planning
- [ ] Set financial goals for next month
- [ ] Schedule upcoming large expenses
- [ ] Review insurance coverage annually
- [ ] Update beneficiaries if life changes occurred
- [ ] Check progress toward annual financial goals
- [ ] Adjust automatic transfers if income changed
