# Website Launch

## Content Review
- [ ] Proofread all pages for typos and grammar
- [ ] Verify all images have alt text
- [ ] Check that all links work (no 404s)
- [ ] Confirm legal pages are in place
  - [ ] Privacy policy
  - [ ] Terms of service
  - [ ] Cookie notice
- [ ] Review content on mobile devices

## Technical Checks
- [ ] Test across major browsers (Chrome, Firefox, Safari, Edge)
- [ ] Verify responsive design at all breakpoints
- [ ] Run PageSpeed Insights and fix critical issues
- [ ] Confirm SSL certificate is installed and forced
- [ ] Test all forms and confirm submissions arrive
- [ ] Verify 301 redirects from old URLs
- [ ] Set up custom 404 error page

## SEO
- [ ] Confirm meta titles and descriptions on all pages
- [ ] Submit XML sitemap to Google Search Console
- [ ] Set up robots.txt
- [ ] Verify canonical URLs are correct
- [ ] Add structured data markup where appropriate
- [ ] Set up Google Analytics and verify tracking

## Infrastructure
- [ ] Configure CDN and caching
- [ ] Set up uptime monitoring
- [ ] Configure automated backups
- [ ] Verify staging environment is separated
- [ ] Review server security settings

## Launch
- [ ] Point DNS to production server
- [ ] Verify site loads on live domain
- [ ] Test all critical user flows one more time
- [ ] Monitor error logs for first 24 hours
- [ ] Announce launch to stakeholders

## Post-Launch
- [ ] Monitor analytics for first week
- [ ] Check Google Search Console for crawl errors
- [ ] Collect initial user feedback
- [ ] Create backlog of post-launch improvements
