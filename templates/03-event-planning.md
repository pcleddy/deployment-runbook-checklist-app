# Event Planning

## Initial Planning (8+ Weeks Out)
- [ ] Define event purpose and goals
- [ ] Set budget and get approval
- [ ] Choose date and time
- [ ] Select and book venue
- [ ] Create guest list
- [ ] Decide on event theme or branding

## Logistics (4-8 Weeks Out)
- [ ] Send save-the-dates or invitations
- [ ] Book catering or food service
- [ ] Arrange audiovisual equipment
- [ ] Plan seating arrangement
- [ ] Hire photographers or videographers
- [ ] Arrange transportation or parking
- [ ] Confirm accessibility requirements

## Content and Program (2-4 Weeks Out)
- [ ] Finalize agenda and run of show
- [ ] Confirm speakers or entertainment
- [ ] Prepare presentation materials
- [ ] Print name badges and signage
- [ ] Create event program or handouts
- [ ] Set up registration system

## Week Before
- [ ] Send reminders to attendees
- [ ] Confirm all vendor deliveries and setup times
- [ ] Do walkthrough of venue
- [ ] Test all AV equipment
- [ ] Brief event staff and volunteers
- [ ] Prepare emergency contact list

## Day Of
- [ ] Arrive early for setup
- [ ] Verify catering and table arrangements
- [ ] Test microphones and projectors
- [ ] Set up registration and check-in table
- [ ] Brief speakers on schedule and logistics
- [ ] Assign someone to handle issues in real time

## Post-Event
- [ ] Send thank-you emails to attendees
- [ ] Collect feedback surveys
- [ ] Pay vendors and close out budget
- [ ] Share photos and recap with team
- [ ] Document lessons learned
