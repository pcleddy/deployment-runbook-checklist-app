# Content Publishing

## Research and Planning
- [ ] Define topic and target audience
- [ ] Research keywords and search intent
- [ ] Review competing content on the topic
- [ ] Create outline with key points
- [ ] Identify sources and data to reference
- [ ] Set target publish date

## Writing
- [ ] Write first draft
- [ ] Include relevant examples and data
- [ ] Add internal and external links
- [ ] Write compelling introduction and conclusion
- [ ] Create section headings for scannability
- [ ] Write meta title and description

## Media
- [ ] Select or create featured image
- [ ] Add images and diagrams throughout
- [ ] Ensure all images have alt text
- [ ] Optimize image file sizes
- [ ] Create social media graphics
- [ ] Record video or audio if applicable

## Editing
- [ ] Self-edit for clarity and flow
- [ ] Check grammar and spelling
- [ ] Verify all facts and statistics
- [ ] Confirm all links work
- [ ] Have a second person review
- [ ] Read aloud for natural tone

## Publishing
- [ ] Format in CMS with proper styles
- [ ] Preview on desktop and mobile
- [ ] Set categories and tags
- [ ] Configure SEO settings
- [ ] Schedule or publish
- [ ] Verify live page renders correctly

## Promotion
- [ ] Share on social media channels
- [ ] Send to email subscribers
- [ ] Post in relevant communities or forums
- [ ] Notify any mentioned sources or collaborators
- [ ] Submit to aggregators or newsletters
- [ ] Monitor comments and engagement
