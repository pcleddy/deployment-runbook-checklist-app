# Wedding Planning

## 6-12 Months Before
- [ ] Set budget and priorities
- [ ] Choose wedding date
- [ ] Book venue for ceremony and reception
- [ ] Hire wedding planner or coordinator
- [ ] Create guest list
- [ ] Book photographer and videographer
- [ ] Start dress or outfit shopping
- [ ] Book officiant

## 3-6 Months Before
- [ ] Send save-the-date cards
- [ ] Book catering and arrange tasting
- [ ] Hire DJ or band
- [ ] Book florist and discuss arrangements
- [ ] Order wedding cake
- [ ] Plan honeymoon and book travel
- [ ] Choose wedding party attire
- [ ] Register for gifts

## 1-3 Months Before
- [ ] Send formal invitations
- [ ] Arrange transportation for wedding day
- [ ] Book hotel room block for guests
- [ ] Schedule hair and makeup trials
- [ ] Plan rehearsal dinner
- [ ] Write vows
- [ ] Get marriage license
- [ ] Finalize seating chart

## Week Before
- [ ] Confirm all vendor details and arrival times
- [ ] Prepare tips and payments for vendors
- [ ] Break in wedding shoes
- [ ] Attend rehearsal and rehearsal dinner
- [ ] Delegate day-of tasks to wedding party
- [ ] Pack for honeymoon
- [ ] Prepare emergency kit (sewing kit, stain remover, etc.)

## Wedding Day
- [ ] Eat breakfast
- [ ] Get hair and makeup done
- [ ] Give gifts to wedding party
- [ ] Arrive at venue for photos
- [ ] Ceremony
- [ ] Sign marriage certificate
- [ ] Reception and celebrate

## After Wedding
- [ ] Send thank-you cards
- [ ] Return rentals (tuxedos, decor, etc.)
- [ ] Preserve wedding dress if desired
- [ ] File name change paperwork if applicable
- [ ] Share photos with family and friends
- [ ] Write reviews for vendors
