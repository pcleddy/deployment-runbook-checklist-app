# New Employee Onboarding

## Before Day One
- [ ] Send welcome email with first-day details
- [ ] Prepare workstation and equipment
- [ ] Set up email account and credentials
- [ ] Create accounts for required software tools
- [ ] Assign onboarding buddy or mentor
- [ ] Schedule orientation meetings

## Day One
- [ ] Office tour and introductions
- [ ] Review employee handbook and policies
- [ ] Complete HR paperwork
  - [ ] Tax forms
  - [ ] Benefits enrollment
  - [ ] Emergency contact information
  - [ ] Direct deposit setup
- [ ] Issue security badge and building access
- [ ] Set up computer and log into all accounts
- [ ] Review team communication channels

## First Week
- [ ] Meet with direct manager for role overview
- [ ] Review team goals and current projects
- [ ] Complete required compliance training
- [ ] Set up development environment (if applicable)
- [ ] Shadow team members on key workflows
- [ ] Schedule recurring one-on-one meetings

## First Month
- [ ] Complete all onboarding training modules
- [ ] Deliver first small assignment or task
- [ ] Attend cross-team introduction meetings
- [ ] Review and understand performance expectations
- [ ] Provide onboarding feedback to HR

## 90-Day Check-In
- [ ] Conduct 90-day performance review
- [ ] Confirm all access and tools are working
- [ ] Address any open questions or concerns
- [ ] Set goals for next quarter
