# Hiring Process

## Job Requisition
- [ ] Define role requirements and responsibilities
- [ ] Set salary range and level
- [ ] Get headcount approval from leadership
- [ ] Write job description
- [ ] Identify required vs. nice-to-have qualifications
- [ ] Define interview panel and process

## Sourcing
- [ ] Post job on company careers page
- [ ] Post to job boards (LinkedIn, Indeed, etc.)
- [ ] Share with employee referral network
- [ ] Reach out to passive candidates
- [ ] Engage recruiting agency if needed
- [ ] Review inbound applications

## Screening
- [ ] Review resumes against requirements
- [ ] Conduct phone screens with top candidates
- [ ] Administer skills assessment or take-home (if applicable)
- [ ] Check for red flags or inconsistencies
- [ ] Move qualified candidates to interview stage
- [ ] Send rejection emails to candidates not advancing

## Interviews
- [ ] Schedule interviews with panel members
- [ ] Send candidates interview details and prep materials
- [ ] Conduct technical or skills-based interviews
- [ ] Conduct culture and values interviews
- [ ] Conduct hiring manager interview
- [ ] Collect structured feedback from all interviewers

## Decision
- [ ] Debrief with interview panel
- [ ] Compare candidates against scorecard
- [ ] Check references for top candidate
- [ ] Make hiring decision
- [ ] Get compensation approval
- [ ] Prepare and send offer letter

## Closing
- [ ] Negotiate offer if needed
- [ ] Get signed offer letter
- [ ] Send rejection emails to other finalists
- [ ] Set start date and notify onboarding team
- [ ] Order equipment and set up accounts
- [ ] Send welcome package
